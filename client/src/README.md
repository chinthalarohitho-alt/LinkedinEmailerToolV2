# client/src/

React application source code.

## Files

| File | Purpose |
|------|---------|
| `main.jsx` | Entry point — mounts `<App />` into the DOM |
| `App.jsx` | Root component — renders 4 tabs (Dashboard, Scraper, Settings, Cron Job) with slide transition animations. Polls cron status to show timer icon when active. |
| `App.css` | Global stylesheet — dark theme with dotted grid background, glass navbar, stat cards, panels, tables, pagination, log viewer, form styles, slide animations. |
